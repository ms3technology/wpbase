# Theme Options List

This document lists the available theme options managed in the `sm_options` table and Redux framework (`opt_name`).

## Core Institute Information
- `institute_name`: Full name of the institute
- `institute_address`: Physical address
- `institute_email`: Contact email address
- `institute_phone`: Contact phone number
- `institute_eiin`: EIIN number
- `institute_code`: Institute Code
- `center_code`: Center Code
- `estd_year`: Establishment Year

## Institution Heads
- `inst_head_title`: Title of the head (e.g., Headmaster, Principal)
- `inst_head_name`: Name of the head
- `homeHeadmasterTitle`: Title for speech/profile on home page
- `headmasterSpeechTitle`: Heading for the speech section
- `homeHeadmaster`: Speech or bio text
- `homeHeadmasterImg`: Profile image URL/Data
- `homeChairmanTitle`: Title for Chairman
- `chairmanSpeechTitle`: Heading for Chairman speech
- `homeChairman`: Chairman speech text
- `homeChairmanImg`: Chairman image URL/Data

## Logos & Media
- `principalSign`: Signature image

## Layout & Content
- `aboutTitelText`: Title for About Us
- `aboutUsText`: About us content (supports HTML)
- `aboutUsTextLimit`: Word/Character limit for homepage display
- `aboutUsMoreBtn`: Text for "Read More" button
- `footerAddress`: Address for footer
- `footerContact`: Contact info for footer
- `footerFbUrl`: Facebook URL
- `copyrightText`: Footer copyright text

## Educational Settings
- `board_name_1`: Primary board name
- `board_name_2`: Secondary board name
- `admitCareNote`: Specialized note for student admit cards


## Identification Prefixes
- `stdidpref`: Prefix for auto-generated student IDs
